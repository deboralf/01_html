1 - O que é HTML?

HTML é uma linguagem de marcação usada no desenvolvimento de sites na internet. Sua sigla significa "Hyper Text Markup Language" , ou seja, linguagem de marcação de hipertexto. Essa linguagem contém tags, atributos e elementos que indicam ações para o navegador interpretar o site.

2 - O que é um elemento HTML? Dê exemplos.

Os elementos HTML ou tags HTML são utilizados para informar ao navegador a configuração e o conteúdo de um site. Cada tag tem um significado semântico (a partir do HTML 5) e é representada por um sinal de "maior que" e "menor que" (<>).

Alguns exemplos são:

<h1>: (Utilizada para títulos, vai até o <h6>)
<p> : (Utilizada na criação de parágrafos)
<br>: (Utilizada para quebras de linha)
<hr>: (Utilizada para criar linhas horizontais)
<tittle>: (Utilizada para definir o título da página, importante para acessibilidade)
<code>(Utilizada para incluir trechos de códigos dentro do HTML)
<link> (Utilizada para vincular links externos como favicons e CSS)

3 - Que ferramentas são necessárias para construir um site HTML ?

Só são necessárias duas ferramentas para construir um site em HTML: Um bloco de notas e um navegador qualquer.

4 - Quais os elementos base necessários para a criação de uma página HTML ?

a estrutura base de um documentos HTML configura-se em : 

<!DOCTYPE html>

<html>

    <head>
    </head>

    <body>
    </body>

</html>

5 - O elemento "br"necessita ser fechado ?

Não, pois trata-se de uma tag vazia.

6 - Para que servem os atributos ? Dê um exemplo.

Os atributos servem para personalizar e configurar as tags. Eles estão contidos dentro das tags e são representados pelo sinal "igual" com "aspas simples ou duplas" (="").

Exemplo:

"src": (muito utilizado na tag <img> para indicar a URL da imagem que o navegador deve exibir).
